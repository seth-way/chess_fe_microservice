### Installation Instructions:
[//]: <> (What steps does a person have to take to get your app cloned down and running?)
To run the current mock WS server in local development, you need to add a .env at the root of the repository with the following variables.
```
RUN_MOCK_SERVER=true
VITE_SOCKET_URL=http://localhost
VITE_SOCKET_PORT=57921
```
